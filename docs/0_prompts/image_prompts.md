## Generation Notes

- **Recommended Tools:** Midjourney, DALL-E 3, or Stable Diffusion XL
- **Format:** Generate as PNG for transparency options, convert to optimized JPG/WebP for production
- **Optimization:** Run through ImageOptim or similar before adding to repo
- **Aspect Ratios:**
  - Hero: 16:9 (1920x1080)
  - Educational: 4:3 (800x600)
